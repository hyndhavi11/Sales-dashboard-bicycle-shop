# Sales Dashboard Analysis (Power BI)

## Project Overview

This project analyzes sales performance using Power BI. The dashboard provides insights into sales trends, product performance, customer contributions, and regional distribution to support business decision-making.

## Tools Used

* Power BI
* Excel / CSV Dataset
* Data Cleaning and Visualization

## Key Features

* Total Sales and Profit KPIs
* Monthly Sales Trend Analysis
* Sales by Product and Segment
* Top Selling Products
* Customer-wise Sales Contribution
* Country-wise Sales Analysis

## Key Insights

* Total sales reached **$118.73M** with a **14.23% profit margin**. 
* **Paseo** is the top-selling product among all products. 
* Sales in **2024 were higher than 2023**, showing business growth. 
* Government and Enterprise segments contribute a significant portion of sales. 

## Project Files

* `Sales_Dashboard.pbix` – Power BI dashboard file
* `Sales_Dashboard_visuals.pdf` – Dashboard preview
* `dataset.csv` – Dataset used for analysis

## Conclusion

This project demonstrates data analysis and visualization skills by transforming raw sales data into actionable insights using Power BI.
